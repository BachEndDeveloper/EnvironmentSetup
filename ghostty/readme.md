# Ghostty config

Location of the file on macOS:
~/.config/ghostty/config

This is the Ghostty terminal configuration. Ghostty replaces iTerm2 in this setup
(the `iterm2/` folder is kept for history only and is no longer used by the Mac script).

## Usage
The Mac setup script copies this file to `~/.config/ghostty/config`. Restart Ghostty (or
reload config) for changes to take effect.

## Font
Uses `MonaspiceNe Nerd Font` (the Nerd-Font-patched Monaspace Neon), installed via
`brew install --cask font-monaspace-nerd-font`.
